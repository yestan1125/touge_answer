题目2: 下列函数的时间复杂度是（ ）。
```c
int func(int n){   
int i=0,sum=0;   
while(sum＜n) 
	sum+=++i;   
	return i;
 } 
```
选项:
A. O(log2n)
B. O(n1/2)
C. O(n)
D. O(nlog2n)
正确答案: B. O(n1/2)

题目8: 下面算法将一维数组a中的n个数逆序存放到原数组中，空间复杂度为（）。
```c
for(i=0;i＜n/2;i++) 
{    t=a[i];
    a[i]=a[n-i-1];
     a[n-i-1]=t;
}  
```
选项:
A. O(n)
B. O(1)
C. O(log₂n)
D. O(n²)
正确答案: B. O(1)
